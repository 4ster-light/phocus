# Phocus

A web blocker for maintaining **Phocus**.

## Usage

1. Clone the repository:

```bash
git clone https://github.com/aster-dev/phocus.git
```

2. Install the dependencies:

```bash
pip install -r requirements.txt
```

3. Run the script:

```bash
python phocus.py
```
And let the program guide you through the process.
